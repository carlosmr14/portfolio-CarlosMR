## Portfolio — Carlos Morales Ruiz

Portfolio personal desarrollado como proyecto de la asignatura de Lenguajes de Marcas del Ciclo Superior de Desarrollo de Aplicaciones Web (DAW).

## Demo

[https://carlosmr14.github.io/portfolio-CarlosMR/](https://carlosmr14.github.io/portfolio-CarlosMR/)

## Descripción

Sitio web estático de cuatro páginas que actúa como carta de presentación profesional. Incluye información sobre mí, mi formación y experiencia, los proyectos que he desarrollado y un formulario de contacto funcional.

## Páginas

- Inicio — Presentación y sección "Sobre mí"
- Currículum — Experiencia en NTT Data, formación DAW y habilidades técnicas
- Proyectos — SuperBros Music Gallery, Base de la Alianza Rebelde y MotoGP Web
- Contacto — Formulario funcional integrado con Formspree

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (vanilla)
- Bootstrap 5 (via CDN)
- Bootstrap Icons
- Google Fonts
- Formspree (formulario de contacto)

## Cómo ejecutarlo en local

1. Clona el repositorio:
```
   git clone https://github.com/carlosmr14/portfolio-CarlosMR.git
```
2. Abre el archivo `index.html` en tu navegador.

No requiere instalar ninguna dependencia ni servidor.

## Funcionalidades JavaScript

- Año del copyright actualizado automáticamente
- Navbar que se compacta al hacer scroll
- Animación de aparición de elementos al hacer scroll
- Botón de volver arriba
